**task03**

Формат для сохранения:  
```json
[
	{
		"data": "тут будет дата новости",
		"href": "тут будет ссылка на страницу новости",
		"title": "тут будет заголовок новости",
		"text": "тут будет текст новости"
	},
	...
]
```
Использовать import json
